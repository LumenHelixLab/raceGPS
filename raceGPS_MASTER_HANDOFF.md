# raceGPS: Real World Driving — Master Handoff v2

**One Earth. Many cities. Many rooms. One player identity. One global career.**

**Core rule:** Cookies handle identity. WebSockets handle the world.

raceGPS is a lightweight, modular multiplayer real-world driving network. Players spawn into cities, cruise mapped environments, use simple room chat, signal nearby drivers, race live or against ghosts, collect route objects, and play fictional Cop-vs-Runner Hot Pursuit.

## Locked Modules

- Global Passport
- World Cells
- Local Rooms
- WebSocket Realtime Layer
- Simple Room Chat
- Cruise Mode
- Signal Challenge System
- Race Mode
- Challenge Mode
- Hot Pursuit Mode
- Ghost Racing
- POI/Route Generator
- Provider Adapter Layer
- Optional Unreal Premium Client Later

## MVP

1. Player creates identity cookie.
2. Player joins a room.
3. WebSocket connects.
4. Player marker appears.
5. Room chat works.
6. Signal challenge broadcasts.
7. Movement packets update map shell.

## Viral Hook

Type a location. Spawn. Drive.

## Safety Boundary

Crime-like gameplay is symbolic: tokens, timers, pings, heat, extraction, capture radius. No real-world crime instruction, police feeds, CCTV, or surveillance integration.
