# Contributing

Thanks for helping build raceGPS.

Good first areas:

- WebSocket message tests
- UI polish
- Route/object templates
- City room presets
- Map adapter experiments
- Documentation
- Graphic/social assets

Rules:

- Do not add secrets.
- Do not scrape map providers.
- Do not add live police/CCTV/surveillance feeds.
- Do not add operational real-crime instructions.
- Keep provider-specific data behind adapters.
- Keep the MVP lightweight and easy to run.
