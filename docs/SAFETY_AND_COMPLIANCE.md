# Safety and Compliance

raceGPS is fictional arcade gameplay. It must not become a reckless-driving tool or real-crime simulator.

Rules:

- No real police/CCTV/surveillance feed integrations.
- No operational crime instructions.
- No reckless real-world driving incentives.
- No map scraping.
- No provider keys in frontend code.
- No restricted provider content committed.
- Real GPS mode, if added, must emphasize legal/safe use.

Crime-like gameplay must remain symbolic: tokens, timers, pings, heat, extraction zones, capture radius.
